# The Analytics Engineering Handbook

This repo has all the resources you need to become an amazing analytics engineer!

## Getting Started

If you are new to analytics engineering, start by understanding what the role entails: analytics engineers sit at the intersection of data engineering and data analysis, building clean, tested, and documented data models that power business decisions.

For more applied learning:
- Check out the [projects](projects.md) section for hands-on examples!
- Check out the [interviews](interviews.md) section for advice on how to pass analytics engineering interviews!
- Check out the [books](books.md) section for a list of high quality analytics engineering books
- Check out the [communities](communities.md) section for a list of high quality analytics engineering communities to join
- Check out the [newsletters](newsletters.md) section to learn via email
